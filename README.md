# Nandan Javagal - Portfolio Website

A modern, responsive portfolio website showcasing AI/ML projects built with Next.js, TypeScript, and Tailwind CSS.
## Quick Start

```bash
npm install
npm run dev
```  

Open http://localhost:3000

## Setup

No backend services are required.
The contact form opens your default email application directly using a `mailto:` link.
## Features
- Dark/Light mode
- AI chat assistant
- GitHub integration
- Responsive design
